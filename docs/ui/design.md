# UI 界面设计文档

**最后更新**: 2026-03-22

## 概述

极简主界面 + 详情弹框的设计模式，让玩家专注于游戏内容。

## 人物详情弹框

### 布局结构

```
┌─────────────────────────────────────────────────────────────────┐
│                        人物详情                        [×]      │
├─────────────────┬─────────────────┬─────────────────────────────┤
│                 │                 │                             │
│   人物立绘       │    基础属性     │       装备精炼              │
│   (大图)         │    (2×3格)     │                             │
│                 │                 │                             │
│   境界/等级      │    装备槽       │   [装备大图]                │
│   突破石数量     │    (2×3格)     │   装备名称/等级             │
│                 │                 │   精炼材料需求              │
│   [境界突破]     │                 │   [精炼] [刷新]             │
│   [自动装备]     │    详细属性     │                             │
│                 │    (显示全部)   │                             │
│                 │                 │                             │
└─────────────────┴─────────────────┴─────────────────────────────┘
```

### 左侧区域：人物立绘
- 尺寸：1:1 正方形
- 境界光环效果
- 底部境界/等级/经验条显示
- 突破石数量
- 境界突破按钮
- 自动装备按钮

### 中间区域：属性 + 装备
**基础属性**（2列3行）：
- 攻击力 | 防御力
- 生命值 | 速度
- 幸运   | 暴击率
- 命中率 | 闪避率

**装备槽**（2列3行）：
- 头盔 | 护甲
- 武器 | 裤子
- 靴子 | 饰品

**点击装备显示详情**：
- 装备名称、等级、品质
- 装备属性列表
- 精炼等级

### 右侧区域：装备精炼
- 装备大图（96×96px）
- 装备名称（品质颜色）
- 精炼等级
- 精炼材料需求
- 精炼按钮
- 刷新按钮

## 交互设计

### 点击头像
- 打开人物详情弹框
- 自动选中第一个装备槽（头盔）
- 更新精炼面板信息

### 点击装备槽
- 高亮选中状态（金色边框）
- 更新右侧精炼面板
- 显示装备详细信息（名称/属性/品质）

### 装备详情显示
- 在中间区域底部显示
- 包含：装备名称、品质、等级、精炼等级、属性列表

---

## 响应式设计

- 桌面：三列布局（立绘 | 属性装备 | 精炼）
- 平板：两列布局（立绘+属性 | 装备+精炼）
- 手机：单列布局（垂直堆叠）

### v1.20 详细更新说明

#### 模态框美化
- **背包模态框**：紫色主题 + 动画图标 + 装饰背景
- **技能树模态框**：响应式设计（w-11/12 max-w-6xl h-[90vh]），保持灵活性
- **世界地图模态框**：统一风格 + 图标化提示栏

#### 功能修复
- **裤子装备合成**：添加`pants`类型到装备检查逻辑
- **技能图片显示**：使用`baseImageId`替代`levels[].imageId`
- **装备合成槽位置**：0%边缘位置（紧凑布局）

#### 设计决策
- 技能树模态框**不使用固定尺寸**，保持响应式以适配不同屏幕
- 装备合成槽位于**0%边缘**，更紧凑地围绕中心结果
