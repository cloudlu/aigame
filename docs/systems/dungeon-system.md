## 🎯 核心变更

### v2.0 更新（2026-03-21）
- ❌ **移除自动采集** - 完全取消挂机资源获取
- ✅ **副本主要来源** - 资源副本成为资源唯一来源
- ✅ **VIP扫荡特权** - VIP玩家可扫荡 + 更多次数
- ✅ **手动战斗优先** - 普通玩家需要手动战斗
- ✅ **配合装备强化** - 消耗匹配装备强化需求

## 概述

资源副本是**唯一的资源获取途径**（除了VIP特权），分为3个难度，每个难度有不同的敌人数量和奖励。玩家需要**主动参与战斗**，VIP玩家可以**扫荡**（自动战斗）。

## VIP特权系统

### 每日挑战次数

| VIP等级 | 每日次数/副本 | 每日总次数 | 扫荡权限 |
|---------|-------------|-----------|---------|
| **VIP 0** | 3次 | 9次 | ❌ 只能手战斗 |
| **VIP 1-2** | 5次 | 15次 | ✅ 可扫荡 |
| **VIP 3-4** | 8次 | 24次 | ✅ 可扫荡 |
| **VIP 5-6** | 12次 | 36次 | ✅ 可扫荡 |
| **VIP 7+** | 20次 | 60次 | ✅ 可扫荡 |

### 扫荡说明

- **扫荡条件**: 通关该难度1次后解锁扫荡
- **扫荡时间**: 立即完成（消耗1次挑战次数）
- **扫荡奖励**: 与手动战斗相同
- **扫荡限制**: 只有VIP1+可以扫荡

## 装备强化消耗匹配

### 强化消耗设计

| 强化等级 | 灵石消耗 | 玄铁消耗 | 成功率 | 累计资源需求 |
|---------|---------|---------|--------|-------------|
| **+1** | 1,000 | 50 | 100% | 1次简单灵石矿 + 1次简单玄铁矿 |
| **+2** | 2,500 | 100 | 95% | 2-3次简单副本 |
| **+3** | 5,000 | 200 | 90% | 5次简单副本 |
| **+4** | 10,000 | 400 | 85% | 10次简单副本 |
| **+5** | 20,000 | 800 | 80% | 20次副本（2-3天） |
| **+6** | 40,000 | 1,500 | 75% | 40次副本（4-5天） |
| **+7** | 80,000 | 3,000 | 70% | 80次副本（1-2周） |
| **+8** | 150,000 | 5,000 | 60% | 150次副本（2-3周） |
| **+9** | 300,000 | 10,000 | 50% | 300次副本（1个月） |
| **+10** | 500,000 | 20,000 | 40% | 500次副本（2个月） |

### 资源平衡计算

**普通玩家（VIP0）**:
- 每日副本次数: 9次（3副本 × 3次）
- 每日资源: 9000灵石 + 390灵草 + 860玄铁（平均）
- 强化+5装备: 需要20天（单件装备）
- 强化+10装备: 需要55天（单件装备）

**VIP玩家（VIP3）**:
- 每日副本次数: 24次
- 每日资源: 24000灵石 + 1040灵草 + 2290玄铁
- 强化+5装备: 需要8天
- 强化+10装备: 需要21天

**结论**: ✅ 资源紧张但合理，VIP有明显优势但不破坏平衡

## 战斗流程

### 手动战斗（VIP0）

```
1. 进入副本 → 选择难度
2. 战斗第1个敌人 → 胜利后自动进入下一个
3. 战斗第2个敌人 → ...
4. 战斗第3/5/7个敌人（Boss）→ 胜利
5. 发放奖励 → 消耗1次挑战次数
6. 评价界面 → 返回副本列表
```

### 扫荡（VIP1+）

```
1. 进入副本 → 选择难度
2. 点击"扫荡"按钮 → 立即完成
3. 显示战斗日志（简化）
4. 发放奖励 → 消耗1次挑战次数
```

## 资源平衡计算（境界加成版）

### 境界收益系数

```javascript
const REALM_BONUSES = {
    0: 1.0,    // 武者境
    1: 1.5,    // 炼气境（+50%）
    2: 2.0,    // 筑基境（+100%）
    3: 3.0,    // 金丹境（+200%）
    4: 5.0,    // 元婴境（+400%）
    5: 10.0    // 化神境（+900%）
};
```

### 实际奖励计算

**公式**：`最终奖励 = 基础奖励 × 境界系数 × (1 + VIP加成)`

**示例（灵石矿脉-困难）**：

| 境界 | 基础奖励 | 境界系数 | VIP0 | VIP3 | VIP7 |
|------|---------|---------|------|------|------|
| 武者 | 10,000 | ×1.0 | 10,000 | 11,000 | 13,000 |
| 炼气 | 10,000 | ×1.5 | 15,000 | 16,500 | 19,500 |
| 筑基 | 10,000 | ×2.0 | 20,000 | 22,000 | 26,000 |
| 金丹 | 10,000 | ×3.0 | 30,000 | 33,000 | 39,000 |
| 元婴 | 10,000 | ×5.0 | 50,000 | 55,000 | 65,000 |
| 化神 | 10,000 | ×10.0 | 100,000 | 110,000 | 130,000 |

**VIP加成**：
- VIP0: 0%
- VIP1-3: +10%
- VIP4-6: +20%
- VIP7+: +30%

### 平衡性结论

**✅ 资源紧张但合理**：
- 武者境玩家需要2-3个月才能强化+10（长期目标）
- 化神境玩家可以更快，但他们有更高的强化需求（化神装备消耗×3）

**✅ VIP有明显优势**：
- VIP3速度提升2-3倍
- VIP7速度提升5-10倍

**✅ 境界成长感强**：
- 每提升一个境界，资源获取速度提升50-100%
- 高境界玩家不会觉得低级副本无意义（依然有可观收益）

**✅ 不会通货膨胀**：
- 高境界装备强化消耗也相应提高（×3倍）
- 实际强化速度依然受限于天数
