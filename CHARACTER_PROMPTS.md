## 📋 文件命名规范

**所有角色图片必须按以下规范命名，存放于 `assets/characters/` 目录**

> **💡 说明**：
> - `Images/` 文件夹用于存放装备、道具、UI等游戏资源
> - `assets/characters/` 专门存放剧情角色立绘，职责分离更清晰

|-----|--------|--------|

**图片要求**：
- 尺寸比例：2:3（竖版）
- 分辨率：8K 或 4K
- 格式：PNG（推荐，支持透明背景）或 JPG
- 构图：半身像或四分之三身像
- 风格：仙侠古风 + 唯美写实

> **💡 提示**：如果AI工具生成的是JPG格式，可以先使用，后续可以用工具去除背景转换为PNG格式。

- **通义万相** (免费) - https://tongyi.aliyun.com/wanxiang
- **文心一格** (免费额度) - https://yige.baidu.com
- **LiblibAI** (免费) - https://www.liblib.art

**国际推荐**：
- **Midjourney** (付费，质量最高)
- **Stable Diffusion** (免费，可本地部署)

### 步骤2：复制中文提示词

1. 选择对应角色的提示词
2. 复制到AI工具
3. 根据需要微调（如年龄、服装颜色等）

### 步骤3：生成并保存

1. 生成图片
2. 选择最佳效果
3. **按规范命名**（重要！）
4. 去背景（可选，建议PNG格式）
5. 保存到 `assets/characters/` 目录

### 步骤4：集成到游戏

在 `game-metadata.js` 中添加图片路径：

```javascript
// 示例：师尊的对话
{
    text: '入门弟子先从基础功法修炼起，等到武者初期修炼有成，便可下山历练。',
    speaker: '师尊',
    speakerImage: 'assets/characters/character_04_master.png'  // ✅ 添加这行
}
```

- 使用相同的种子值（seed）

- 保持相同的关键词：`仙侠古风立绘`、`唯美写实`、`8K高清`

- 统一调整色调

```javascript
// mainQuest.js - nextStoryPage() 函数中添加
function nextStoryPage() {
    const page = storyPages[currentPage];

    // 显示文字
    document.getElementById('story-text').textContent = page.text;
    document.getElementById('story-speaker').textContent = `【${page.speaker}】`;

    // ✅ 显示角色立绘
    const speakerImage = document.getElementById('story-speaker-image');
    if (page.speakerImage) {
        speakerImage.src = page.speakerImage;
        speakerImage.style.display = 'block';
        speakerImage.style.animation = 'fadeIn 0.5s';  // 淡入动画
    } else {
        speakerImage.style.display = 'none';
    }
}
```
